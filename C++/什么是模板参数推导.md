模板参数推导很简单，就是C++里面可以定义模板类或者模板函数，一般来说使用模板类或者模板函数的时候需要**明确给定模板参数类型**。比如`vector<int>` 这里的`int` 就是明确给出的模板参数类型(argument)。

但有时候为了代码的整洁度，我们会**省略模板参数类型**。这时候就需要编译器根据给定的实参**自动的推导出模板参数类型**。比如有下面的简单函数模板：

```c
template <typename T>
void print(T value) {
    std::cout << value << std::endl;
}
```

当我们调用这个函数时，可能会这样：

```c
print(42);       // T is deduced to be int
print(3.14);     // T is deduced to be double
print("Hello");  // T is deduced to be const char*
```

而不是

```c
print<int>(42);    
print<double>(3.14);     
print<string>("Hello");
```

其中就是利用了编译器的模板参数推导功能，根据给定的实参，自动推导出了模板参数类型。(是的就是这么简单)

# What is template argument deduction？

Template argument deduction is a feature in C++ that allows the compiler to automatically determine the types of the template parameters based on the arguments passed to the template function or class. This helps in reducing the amount of code needed to explicitly specify template arguments, making the code cleaner and easier to maintain.

Here’s a more detailed explanation:

### Basics of Template Argument Deduction：

When you define a template function or class, you specify one or more template parameters. When you call a template function or instantiate a template class, you typically provide arguments that correspond to these parameters. Template argument deduction is the process by which the compiler determines the appropriate template parameters based on these arguments.

In these above calls, the compiler examines the type of the argument passed to `print` and deduces the template parameter `T` accordingly.
