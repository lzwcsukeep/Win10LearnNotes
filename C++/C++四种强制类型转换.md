# static_cast

- static_cast做了什么？什么时候用？

![](..\img_src\2024-05-16-11-52-25-image.png)

- `static_cast`在继承体系下面向下或者向上转型的时候会在子类对象内部移动指针。使得指针指向对应的父类的位置。

![](..\img_src\2024-05-16-11-48-37-image.png)

- `static_cast` 在向下转型的时候，如果是不同子类之间转换，并不会给出错误提示。(之前你自己试验过了的)

![](..\img_src\2024-05-16-12-02-36-image.png)
