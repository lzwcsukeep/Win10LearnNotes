### 如何让make 只打印编译命令而不进行实际的编译行为？

答： 使用 -n 选项即可完美匹配你的需求。

要让 make 只打印编译命令而不执行实际编译，可以使用 make 的 -n（--just-print）选项，它会模拟执行编译过程并输出所有要执行的命令，但不会真正运行这些命令。

示例： make -n  # 打印编译命令不进行编译
make clean -n  # 查看make clean 的命令行为，不进行clean操作。

--------------------
扩展：其他相关选项
-p 选项：除了打印命令，还会输出 make 内部的变量和规则（适合调试复杂 Makefile）：
使用示例:
make -p  # 打印命令、变量和规则，不执行编译

-q 选项：仅检查目标是否需要更新，不输出命令也不执行（返回 0 表示无需更新，1 表示需要更新）：
使用示例:
make -q  # 静默检查，无输出，仅返回状态码

总结
make -n 是最直接的方式，它会忠实模拟编译过程并输出所有要执行的命令，但不会产生任何实际文件或修改系统状态，
非常适合验证 Makefile 的命令是否符合预期（如编译选项、路径是否正确）。
