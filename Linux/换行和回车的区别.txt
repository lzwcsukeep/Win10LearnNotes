`\n` : line feed ,newLine

`\r`: Carriage Return  ,called ^M, CR

换行的字符表示是 `\n` ,回车的字符表示是 `\r`

在Linux 里面只需要`\n` 就可以表示换一行了，但是windows 下需要 `\r\n` 才表示换行。

原因和历史上的打字机有关,细究起来，在老式的物理打字机中换行仅仅把光标移动到下一行，但是不一定到行的最左侧。回车是把光标移动到行的最左侧。
看下面例子，如果我在单词 now 后只敲击换行`\n`，是会下面效果的，只有紧接着敲击回车`\r`，才会真正的从下一行开始。
```
I will hit enter now
                                 but I forgot to return the carriage
```
当然，这是老式的打字机，现在都是电子打字，其实只需要`\n` 一个字符表征另起一个新行就行。

总结：

细究起来，在老式的物理打字机上
`\n` 表示换行，光标不移动到行首
`\r` 表示光标移动到行首。
因此以前真正表示另起一行需要按两下"回车(\r)"以及"换行(\n)"。
但现在其实只需要一个字符`\n` 就行。
