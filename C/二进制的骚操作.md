# n & (n - 1)

对于一个整数 n, 执行n-1的效果：

该整数n的二进制表示中，`n - 1`计算的效果是n的二进制表示中从右往左数第一个1(最低位的1)变成0，而这个1右边的所有0变成1。不信自己随便用个数字试试。同时这个1左边的所有数字保持不变。

如果此时用 `n-1` 与原来的`n`做按位与(&)操作，那么就是把n的`二进制表示`中`最右侧`的`1`变为`0`，而`其他任何位`都`保持不变`。

- 结论： 整数 n ， n & ( n - 1) 的效果为：

> 把n的`二进制表示`中`最右侧`的`1`变为`0`，而`其他任何位`都`保持不变`。
> 
> 示例： n = 100111000  ---》   n & (n-1)  = 100110000
