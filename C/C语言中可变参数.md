# 定义可变参数函数
函数定义参数列表里面使用 `...` ，省略号 ... 就代表可变参数的列表。注意是列表,list

然后在函数内部获取这个可变参数的列表：
使用 va_list 类型的变量，如 va_list ap ;

使用 va_start 确定可变参数的列表的起始位置。如：

```c
    va_list ap;  // 定义可变参数列表

    // 初始化参数列表（第二个参数为最后一个固定参数）
    va_start(ap, fmt);
    // ap 是可变参数的列表。
```

注意： 
1. 可变参数函数之中， ... 就代表的是一个列表，这个列表的元素是传进来的多个可变参数。
2. 可变参数函数内部，使用 va_list 类型的变量，指向(确定)可变参数组成的列表的位置。
